# ADR-003 Use Streamlit


## Status

Accepted


## Context

项目核心价值：

- Agent 能力
- Memory 系统
- RAG 流程


不是复杂前端。


MVP 阶段需要快速验证。


---

## Decision

使用 Streamlit 作为 MVP 前端。


实现：

- Chat UI
- Memory 查询
- 项目知识展示


---

## Alternatives


### React + Tailwind + Ant Design


Rejected for MVP。


原因：

增加前端开发成本，降低 AI 能力验证速度。


---

+
